# 🌾 Rice Mill Sales Dashboard (2025-2026)

## 📊 Project Overview
Interactive Excel dashboard analyzing Rice Mill sales data 
for the year 2025-2026 using Pivot Tables, Charts & Slicers.

## 🔍 Key Insights
- 💰 Total Sales Amount: ₹4,10,70,181.95
- ⚖️ Total Quantity Sold: 16,148.27 Quintal
- 🏆 Highest Sales Month: April (₹41,36,640.2)
- 🌾 Top Commodity: Konda Polish (highest quantity)
- 🗺️ Top State by Quantity: MP (Madhya Pradesh)
- 🍚 Rice Varieties Tracked: KRANTI, PARMAL, TAJ
- 📦 Bag Sizes: 26kg, 30kg, 50kg

## ❓ Business Questions Answered
1. Which month had the highest sales?
   → April with ₹41,36,640
2. Which commodity was sold the most?
   → Konda Polish
3. Which state purchased the most quantity?
   → Madhya Pradesh (MP)
4. Which rice variety is most popular?
   → KRANTI & PARMAL (dominant in all states)

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Slicers for interactivity
